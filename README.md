

# Student Performance Prediction

This project is part of **Main Assignment 1: ML Model Deployment**.

It predicts whether a student will **Pass or Fail** based on:
- Study Hours
- Attendance
- Previous Score

The model is built using **Logistic Regression (Scikit-learn)** and deployed using **Streamlit**.

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
